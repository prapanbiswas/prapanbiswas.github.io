How to Use This Blog Demo

December 04, 2025 1 min read

Welcome to your new blog demo! This file is designed to help you create new blog posts easily.

How to Add Images

We have created a dedicated folder for your blog images at blog-posts/images/. When you want to add an image to your blog post, simply save it in that folder and use the following HTML code:

<img src="images/your-photo.jpg" alt="Description">

This keeps your project organized and ensures images load correctly.

Editing Content

You can edit this text just like any other HTML file. Use <h2> for main headings and <h3> for subheadings.

  • Keep your paragraphs short and readable.
  • Use lists to break up information.
  • Don't forget to update blog-data.json to make this post appear on the homepage!
